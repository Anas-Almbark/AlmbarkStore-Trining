## Key Features

### Product Management

- Display products in a responsive and attractive grid
- Product details with high-quality images
- Product ratings and reviews
- Add new products with user-friendly interface

### Search & Filtering

- Filter by categories
- Sort products (price, rating, name)
- Dynamic search results

### Shopping Cart

- Add and remove products from cart
- Update quantities easily
- Automatic total calculation
- Save cart to local storage
- Interactive dropdown cart interface

### Responsive Design

- Compatible with all screen sizes
- Mobile-first design approach
- Responsive navigation menu for mobile
- Optimized user experience across all devices

### User Interface

- Modern design using Tailwind CSS
- Smooth animations and transitions
- Font Awesome icons

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **Tailwind CSS**
- **Font Awesome**
- **Fake Store API**
